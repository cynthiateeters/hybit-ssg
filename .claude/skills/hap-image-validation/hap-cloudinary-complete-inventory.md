## How to construct Cloudinary URLs

**Standard format**:

```
https://res.cloudinary.com/cynthia-teeters/image/upload/f_auto,q_auto,w_[WIDTH],c_limit/v[VERSION]/[FILENAME].jpg
```

**Required components**:

- `f_auto` - Automatic format (WebP when supported)
- `q_auto` - Automatic quality optimization
- `w_[WIDTH]` - Width constraint (80, 100, 120, 150, or 200)
- `c_limit` - Limit scaling to prevent upscaling
- `v[VERSION]` - Version number (all verified in this inventory)
- `[FILENAME]` - Exact filename from list below

**Width guidelines**:

- Hero images: `w_200`
- Section images: `w_150`
- HAP note callouts: `w_150`
- Footer images: `w_80` or `w_120`
- Small inline: `w_100`

## Usage guidelines

### Emotional journey mapping

**Typical learning lab progression**:

1. **Hero/Introduction**: `hap-laptop` or `hap-waving`
2. **Confusion/Mistakes**: `hap-confused-map` or `hap-broke-things`
3. **Breakthrough**: `hap-brain-explodes` or `hap-celebrating`
4. **Understanding/Practice**: `HAP-learner` or `hap-thumbs-up`
5. **Closing Reflection**: `HAP-learner`
6. **Footer**: `HAP-learner`

### Blog post emotional journey

**Typical blog post progression**:

1. **Opening**: `hap-laptop` (focused on new topic)
2. **Why this topic**: `hap-confused-map` (initial confusion)
3. **What I built**: Screenshot of lab (not HAP pose)
4. **Biggest challenge**: `hap-broke-things` (disaster story)
5. **What Prof. Teeters taught me**: `hap-brain-explodes` (mind-blowing lesson)
6. **Try it out**: `hap-waving` (friendly farewell)

### Distribution recommendations

**Per station (6-7 total images)**:

- 1× Hero (laptop or waving)
- 1-2× Confusion/struggle (confused-map, broke-things, sconcerned-laptop)
- 1× Major breakthrough (brain-explodes or celebrating)
- 2-3× Practice/understanding (HAP-learner, thumbs-up)
- 1× Footer (HAP-learner)

**Per blog post (3-5 HAP poses)**:

- 1× Hero (laptop - 200px)
- 1× Confusion (confused-map or broke-things - 150px)
- 1× Breakthrough (brain-explodes - 150px)
- 1× Farewell (waving - 120px)
- 1-2× Screenshots of learning lab (not HAP poses)

### Reserve for special moments

**Don't overuse**:

- `hap-brain-explodes` - Save for TRUE paradigm shifts (max 1 per station, 1 per blog post)
- `hap-celebrating` - Save for major victories (end of station or challenge)
- `hap-broke-things` - Save for significant mistakes (not every error)
